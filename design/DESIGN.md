# Design System — Finance Dashboard

## Paleta

| Uso | Cor | Hex |
|---|---|---|
| Fundo | Roxo-navy escuro | #15121F |
| Card | Roxo mais claro | #1E1A2B |
| Texto principal | Quase branco | #F5F3F7 |
| Texto secundário | Cinza-roxo | #9D94B0 |
| Receita | Verde esmeralda | #34D399 |
| Despesa | Coral/pink | #F87171 |
| Destaque | Pink vibrante | #EC4899 |

## Tipografia
- Display (números dos KPIs): a definir
- Corpo/tabela: a definir (precisa ser bem legível em números)

## Glassmorphism — regras
- Fundo semi-transparente: `bg-white/5` ou `bg-white/10`
- Blur: `backdrop-blur-md`
- Borda sutil: `border border-white/10`
- Sombra: `shadow-xl shadow-black/20`
- Cantos: `rounded-2xl`

## Referências
- Helios Investments (roxo escuro + glow rosa) → paleta principal
- lumify/Nexus → estrutura dos cards e tipografia de título
- Dashboard verde → padrão "vs mês anterior" + donut chart
